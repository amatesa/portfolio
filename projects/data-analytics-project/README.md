# Data Analytics Case Study

Sitio estático para GitHub Pages basado en el notebook `Trabajo_Practico_Integrador_Data_Analytics.ipynb`.

## Fuente de datos

El notebook es la fuente de verdad del contenido, métricas y conclusiones del sitio. El análisis trabaja con tres datasets:

- `ventas.csv`
- `clientes.csv`
- `marketing.csv`

## Métricas principales extraídas del notebook

- Ventas limpias: 2,998
- Ingreso total limpio: $1,467,093.52
- Venta promedio: $489.36
- Mediana de ventas: $418.06
- Producto con mayor ingreso: Lámpara de mesa
- Categoría con mayor ingreso: Electrodomésticos
- Cantidad de productos: 30
- Cantidad de categorías: 3
