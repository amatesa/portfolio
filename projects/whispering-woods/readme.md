Whispering Woods
Short description

Whispering Woods is a 2D pixel-art platformer developed as the final project for the Programming I course in the Video Game Development and Production program at UTN Buenos Aires.

Players control Ren, a fox trapped inside a cursed forest where an ancient corruption has twisted the environment. The objective is to explore interconnected areas, collect mystical keys, avoid hostile creatures, and escape before the corruption consumes the forest.

The project focuses on gameplay programming, modular architecture, state management, audio interaction, and Unity best practices while implementing a complete playable experience. It applies the programming concepts introduced throughout the course, including C#, Unity scripting, object-oriented programming, game loops, collision systems, UI, scene management, and project publishing.
